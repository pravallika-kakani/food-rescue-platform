# 🌿 Food Rescue Platform

A web application to reduce food waste by connecting food donors with volunteers.

## 🔗 Live Demo
👉 https://food-rescue-platform-17rs.onrender.com/

## Features
- Donate surplus food from restaurants or events
- Volunteers can claim food and earn reward points
- Leaderboard to track top volunteers
- Redeem points for vouchers and rewards

## Tech Stack
- Python / Flask
- SQLite
- HTML, CSS, JavaScript
- Deployed on Render

## How to Run Locally
pip install -r requirements.txt
python app.py
